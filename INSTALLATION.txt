QUICK START - 3 STEPS:
----------------------

1. INSTALL DEPENDENCIES (2 minutes)
   
   Open terminal/command prompt in this folder and run:
   
   npm install
   
   (Wait for installation to complete)


2. CREATE ENVIRONMENT FILE (30 seconds)
   
   Windows:
   copy .env.local.example .env.local
   
   Mac/Linux:
   cp .env.local.example .env.local


3. START DEVELOPMENT SERVER (30 seconds)
   
   npm run dev
   
   Then open browser to: http://localhost:3000


SUCCESS! Your website is now running!

================================================================================

NEXT STEPS:
-----------

1. Update your company information in .env.local

2. Customize the content:
   - Homepage: pages/index.tsx
   - About: pages/about.tsx
   - Services: pages/services.tsx
   - Projects: pages/projects.tsx
   - Contact: pages/contact.tsx

3. Add your images to: public/images/

4. Change your brand color in: tailwind.config.ts
